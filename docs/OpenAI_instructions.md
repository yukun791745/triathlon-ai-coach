【生理学基礎知識】

■VO2max（最大酸素摂取量）
- 持久力の指標。エリート男性70-85ml/kg/min、女性60-75ml/kg/min、一般40-50
- 遺伝的上限あり（トレーニングで10-20%改善可能）
- 計算式：VO2max = 15.3 × (HRmax/HRrest)
- Cooper Test: VO2max = (距離m - 504.9) / 44.73

■乳酸閾値（LT/LTHR）
- LT1: 血中乳酸2mmol/L、有酸素ベース。HRmax65-75%
- LT2/OBLA: 血中乳酸4mmol/L、持続可能最大強度。HRmax80-88%
- FTP（機能的閾値パワー）: 1時間維持可能な最大パワー
- LT向上トレーニング: テンポ走、クルーズインターバル

■心拍ゾーン（5ゾーンモデル）
- Z1回復(50-60%HRmax): 疲労回復、脂肪燃焼
- Z2有酸素(60-70%): ベース構築、ミトコンドリア増加
- Z3テンポ(70-80%): LT1付近、持久力向上
- Z4閾値(80-90%): LT2付近、レースペース
- Z5無酸素(90-100%): VO2max向上、短時間高強度

■ランニングエコノミー（RE）
- 同じ速度での酸素消費量。低いほど効率的
- 改善要因: ストライド最適化、接地時間短縮、垂直振動低減
- 週1回のストライド走（100m×6-8本）が効果的

【トレーニング理論】

■ピリオダイゼーション
- 基礎期(12-16週): Z2中心、週10-15時間、ベース構築
- 構築期(8-12週): 強度増加、インターバル導入
- 専門期(4-8週): レース特異性、ブリック増加
- テーパー期(2-3週): 量60-70%減、強度維持
- 回復期(1-2週): アクティブリカバリー

■超回復理論
- 適切な負荷→疲労→回復→超回復（48-72時間）
- オーバートレーニング指標: 安静時HR上昇、HRV低下、睡眠障害

■トレーニング負荷管理
- TSS（Training Stress Score）= 時間×IF²×100（IF=強度係数）
- CTL（フィットネス）: 42日間TSS移動平均
- ATL（疲労）: 7日間TSS移動平均
- TSB（フォーム）= CTL - ATL（レース時+15~+25が理想）
- 週間TSS増加は10%以下に抑制

■HIIT vs 低強度持久走
- HIIT: VO2max向上、時間効率高、回復に48-72時間必要
- 低強度（80/20ルール）: 80%をZ2、20%をZ4-5に
- ポラライズドトレーニングが長距離で最も効果的

【発汗量測定・水分補給】

■体重変化法（ゴールドスタンダード）
計算式:
発汗量(L) = (運動前体重 - 運動後体重) + 摂取水分量 - 尿量
※1kg ≒ 1L
発汗率(L/h) = 発汗量 / 運動時間(h)

■補給設計の目安
- 体重減少を2%未満に抑制
- 摂取量 = 発汗率 × 運動時間 × 0.6-0.8（胃腸耐性に応じ調整）
- Na補給: 500-1000mg/時間（発汗量・塩分濃度による）

■発汗率の典型値
- 軽度運動・涼しい環境: 0.3-0.5 L/h
- 中強度・温暖環境: 0.5-1.0 L/h
- 高強度・暑熱環境: 1.0-2.5 L/h
- エリートアスリート最大: 3.0 L/h以上
